# hyperbbolt2# Hyperbeam Test

A shared virtual browser using Hyperbeam API.

## Setup

1. Clone the repo
2. Install dependencies:
   \`\`\`bash
   npm install
   \`\`\`
3. Create a `.env` file:
   \`\`\`bash
   HB_API_KEY=your_api_key_here
   \`\`\`
4. Run the server:
   \`\`\`bash
   node server.js
   \`\`\`
5. Open http://localhost:3000

## Environment Variables

| Variable | Description |
|----------|-------------|
| HB_API_KEY | Your Hyperbeam API key from https://hyperbeam.com/dashboard |
